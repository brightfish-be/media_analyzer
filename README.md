## Installation

You can install the package via composer:

```bash
composer require cinemapub/spx_media_analyzer
```

## Usage

``` php
$obj = new Cinemapub\SpxMediaAnalyzer();
echo $obj->echoPhrase('Hello, cinemapub!');
```
